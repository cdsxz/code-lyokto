# code-lyokto

**Exploring Yocto from scratch — a journey to defeat IA.**

## Context

I'm an embedded developer but never seriously touched Linux distro building.
And every time I think about learning something new, my brain whispers: "It's 2026, LLMs will handle this anyway."

This repo is my Lyoko—a deliberate journey into Yocto, documented as it unfolds. Not because I think I'll "beat" AI at recipe generation, but because **understanding** embedded Linux means something fundamentally different than **producing** it.

(Yes, the name is a cheeky nod to Code Lyoko, my favorite series as a kid.)

## About This Repository

This repository is my travel log exploring Yocto and all my thoughts around it. I would be happy if someone could find any interest in it, whether for motivation or information.

## Goals
Goals will change as I learn, because I won't follow a tutorial — just my curiosity.

- [ ] Build custom Yocto distributions
